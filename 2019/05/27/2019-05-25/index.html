SEO优化

发表于 2019-05-27

###时间:2019-05-25

1.百度抓取标题一定是从h1->h2->h3…
2.请求的时候尽量用get请求而不是用post请求,这里由于百度抓取的时候是很难抓取或者根本抓取不到post请求的内容.
3.页面的路径要用绝对路径,而相对路径百度是抓取不到的
4.在一个网站里放一个xml文件,把网站所有页面的绝对路径存放到里面有利于搜索引擎抓取.
5.如果你不想让百度爬虫爬去, 而想要他只能站内搜索,可以添加一个js文件,来告诉百度爬虫,不让你爬去我网站里面的内容.例如淘宝内的商品只能在淘宝内搜索.
